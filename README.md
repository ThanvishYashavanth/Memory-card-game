# Memory Card Game

## How to Play
1. Click on a card to flip it.
2. Flip two cards at a time to find matching pairs.
3. If the cards match, they remain face up.
4. If they do not match, they flip back.
5. The game ends when all pairs are matched.

## Features
- Responsive design (desktop & mobile)
- Smooth card flip animations
- Move counter
- Timer
- Proper card shuffling (Fisher-Yates algorithm)
- Restart game button

## Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript
